## Installation

Install the project in editable mode:

pip install -e .

## Configuration

1. Copy the example environment file:
   cp .env.example .env
2. Update the `.env` file with the required values.

https://docs.litellm.ai/docs/providers (list of supported models)
## Usage

Run the agent:

agent
